# PersonalTrainingDiary

<h3>FUNCTIONALITY</h3>
<h5>Personal Training Diary is a web application designed for personal trainers.</h5>
<p>
<b>Functionality:</b>
- register trainers,
- login trainers,
- add / modify / delete clients
- show all clients

- (in the future: register and login clients)
- (in the future: booking training).

</p>

<b>Technologies:</b>
- Java, 
- MYSQL,
- Hibernate,
- JSP,
- Spring,
- Spring Data,
- Spring Security,
- MVC, (Maven Project).
