# PersonalTrainingDiary

<h2>FUNCTIONALITY</h2>
Personal Training Diary is a web application designed for personal trainers.
<p>
Functionality:

- register trainers,
- login trainers,
- add / modify / delete clients
- show all clients

- (in the future: register and login clients)
- (in the future: booking training).

</p>

Technologies:

- Java, 
- MYSQL,
- Hibernate,
- JSP,
- Spring,
- Spring Data,
- Spring Security,
- MVC, (Maven Project).
